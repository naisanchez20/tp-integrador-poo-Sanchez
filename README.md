#Sistema de Catalogo de Animes


##Como Ejecutar el programa:


Opcion uno:
Desde el IDE ejecutar el archivo de la clase 'PanelPrincipal.java'


Opcion dos:
1. Se puede compilar el proyecto: 
```bash
javac -d bin -sourcepath src src/ui/PanelPrincipal.java
```
2. Luego ejecutar:
```bash
java -cp bin ui.PanelPrincipal.java
```
